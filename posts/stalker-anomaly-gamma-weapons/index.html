旧バージョンGAMMAプレイヤー向け注意喚起
旧バージョンの一部修理アイテムはクラフトアイテムになりました。銃の全体耐久値という概念もなくなりました。

銃の修理とは
#

銃の修理とは銃に取り付けられている部品の耐久値を上げることで弾詰まり発生率を0%に近づけダメージとアーマー貫通を示すDMG&APを100%にすることが目的です。

すべての部品耐久値を100％にしても弾詰まりが0%にならない場合はサプレッサーを外してみてください。

銃と銃の部品
#

銃は銃と銃の部品に分かれています。

銃
#

銃は主にNPCの死体か隠し場所（Stash）から入手できます。
左下は銃の部品耐久値で部品がない場合は灰色、1～20%は赤、21～39%は橙色、40～59%は黄色、60～100%は緑で表示され、右下が武器タイプでA～Dのタイプがあります。

銃に取り付けている部品の耐久値と名称は詳細から確認できます。

銃の部品
#

銃の部品は主に銃を分解、フィールドストリップすることで入手できます。

  • フィールドストリップは銃を破壊せずにバレル部品以外の部品を取り外すことができます。
  • 分解はマルチツールが必要で銃を破壊しすべての部品を取り外すことができます。

銃にどの部品を取り付けることができるか確認したい場合は銃にマウスをホバーさせることで、対応する部品がハイライト表示されます。

修理アイテム
#

銃の部品を修理するにはその部品に合ったタイプの修理キットまたはクリーニングキットが必要です。

  • 修理キットは価格は高いが部品耐久値が1%以上あれば修理でき、部品交換にも利用します。
  • クリーニングキットは価格は安いが部品耐久値が60%以上必要です。

銃の部品がどのタイプの修理アイテムに該当するかは詳細から確認できます。

必ずしも銃本体のタイプとそれに取り付ける部品の修理アイテムタイプが一致するとは限りません。
SKSの武器タイプはBですが、SKSの部品であるバレルの修理アイテムタイプはDです。

クラフトレシピ
#

修理アイテムのクラフトレシピです。

修理
#

修理には一般的な部品修理と部品を取り外さずに修理するメンテナンスがあります。

部品修理
#

修理キットまたはクリーニングキットを使用でインタラクトできます。
部品修理は現バージョン上必ず100%になるので補助材料を選択する必要はありません。

メンテナンス
#

銃と同じタイプの修理キット、クリーニングキットを持っている状態で銃を右クリックするとメンテナンスできます。
「交換」なら修理キットを、「洗浄」ならクリーニングキットを消費します。

部品の取り付けと交換
#

バレル以外の部品はドラッグアンドドロップで取り付けと交換できます。

バレルを含む銃の部品交換にはツールまたは携帯型ワークショップと交換したい銃と同じ武器タイプの修理キットが必要です。
ツールを右クリックから作業台を開くを選択しウィンドウを開き上部タブの🔧から銃部品の交換をします。

まとめ
#

  • 銃は銃と銃の部品に分かれていて銃の部品を修理交換することで弾詰まりなく最大ダメージを保証できる。
  • 銃の部品耐久値が60%以上ならクリーニングキット、60%未満なら修理キットを使い、耐久値を100%にする。
  • バレル交換には修理キットとツールが必要（修理キットの使用回数は消費しない）
